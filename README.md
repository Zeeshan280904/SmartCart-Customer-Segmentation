# SmartCart Clustering System (Unsupervised Learning)

This project applies unsupervised machine learning techniques to segment customers of an e-commerce platform based on their demographics, purchasing behavior, and engagement patterns. The goal is to discover meaningful customer groups that can help businesses design personalized marketing strategies and improve customer retention. 

The dataset consists of 2240 customers with 22 attributes, including demographic details, spending on different product categories, purchase frequency, and website activity. After performing data cleaning, feature engineering, encoding, and scaling, dimensionality reduction was applied using PCA for visualization and noise reduction.

Customer groups were identified using K-Means and Agglomerative Clustering, with the optimal number of clusters determined through the Elbow Method and Silhouette Score. The resulting clusters reveal patterns in customer spending habits, income levels, and engagement, enabling data-driven decision making for targeted marketing and customer relationship management.
